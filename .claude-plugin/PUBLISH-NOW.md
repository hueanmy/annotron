# 🚀 Publish Annotron Now - 3 Steps

## Only 1 Thing Left: Convert Icon SVG → PNG

### Quick Method (30 seconds)

**Using Online Converter:**

1. Go to: https://convertio.co/svg-png/
2. Drag & drop: `icon.svg`
3. Download as PNG
4. Save to: `.claude-plugin/icon.png`

Done! ✅

## Now Submit (Choose One or Both)

### Option 1: Claude.ai Plugin Marketplace

1. Go to: **https://claude.ai/plugins**
2. Click: **"Submit a Plugin"**
3. Fill in:
   - Name: `Annotron`
   - Description: `Point-and-click annotation editor for AI artifacts. 3-5 second feedback loops. Beautiful HTML rendering. No copy-paste.`
   - Icon: Upload `icon.png`
   - Repository: `https://github.com/hueanmy/annotron`
   - Category: `Productivity`
   - Pricing: `Free`
4. Click: **"Submit for Review"**
5. Wait 3-5 days ⏳

---

### Option 2: Claude Code Registry

1. Go to: **https://code.claude.ai/plugins**
2. Click: **"Publish Plugin"**
3. Connect GitHub or upload files:
   - `.claude-plugin/plugin.json`
   - `.claude-plugin/icon.png`
   - `.claude-plugin/README.md`
4. Click: **"Publish"**
5. Wait 1-3 days ⏳
