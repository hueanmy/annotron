PRD — Bulk Asset Review Queue

Epic ID: CF-482   Author: Product Owner   Status: In Review   Created: Jul 18, 2026

1. Problem Statement

Retouchers lose ~40 minutes per shoot switching between tools to review and approve product images. There is no single queue, so approvals get lost across shoots and deadlines slip.

Describe the user or business problem this epic solves. Why does it exist?

2. Goals

  • Cut review time per shoot by 50%
  • Single approval queue across all active shoots
  • Keyboard-first navigation through the queue

3. Non-Goals

  • Out of scope: mobile capture from the review tool
  • Will not address: DAM migration

4. Affected Areas

AreaChangeNotes
Review toolNew queue view + approval controlsPrimary surface
NotificationsApproval-request eventsReuses existing bus
ReportingTurnaround-time metricsNew dashboard tile

5. User Stories

  • As a retoucher, I want a single queue so that I stop losing approvals across shoots.
  • As a lead, I want turnaround metrics so that I can spot bottlenecks early.
