## GitHub Release Notes / Discussions

### GitHub Release Description
```
# annotron v1.0 — AI Artifact Review Made Simple

**What's new:**

✨ Point-and-click annotations for HTML and Markdown artifacts
🤖 Auto-apply feedback loop (integrated with Claude Code)
📋 Outline navigation for long documents
💬 Annotation threads with image attachments
⚡ Live Markdown editor with diagram support
🔒 Local-only, zero setup

**Why it matters:**

Feedback doesn't have to be slow. Instead of describing changes:
- Click an element
- Add a note  
- AI applies it instantly
- See the result in your browser

**Get started:**

```bash
npm install -g annotron
annotron my-file.md --agent
```

**Demo:** youtube.com/@studyingwithmeii
**Docs:** github.com/hueanmy/annotron

Special thanks to the Claude Code team for inspiration on loop engineering.

---

Made with ❤️ for better AI-human collaboration
```
