## 🎬 Animation Style Guidelines

### Drawing Animation
- **Hand-drawn effect**: Elements "draw themselves" with pen stroke animation
- **Line thickness**: Varies slightly (realistic hand drawing)
- **Timing**: 0.3-0.5 seconds per element draw
- **Easing**: Ease-out for drawing, ease-in-out for movements

### Icon Animation
- **Bounce**: Icons bounce slightly when appearing (elastic)
- **Fade**: Smooth fade-in/fade-out between scenes
- **Ripple**: Click effects create expanding circular ripples
- **Shake**: Refresh/loading moments have gentle shake

### Transitions
- **Between scenes**: Fade to white, then draw new scene (2-3 seconds)
- **Within scene**: Smooth fade/slide transitions (0.5-1 second)
- **Wavy lines**: Represent communication/data flow

### Sound Design (Optional)
- Pen scratching sounds 🖊️ (light, not annoying)
- Subtle "pop" when comment appears 💬
- Soft "whoosh" when data flies between agent/browser
- Brief "ding!" when changes apply
- Upbeat royalty-free background music (lo-fi, friendly tone)

## 📌 Notes for Animator

1. **Keep it simple** - avoid over-complicated UI details, focus on concept
2. **Make icons cute** - rounded shapes, friendly expressions
3. **Use consistent line weight** - hand-drawn feel should be uniform
4. **Add personality** - little bounces, wiggles, facial expressions on icons
5. **Text legibility** - keep text large and readable
6. **Pacing** - not too fast, let viewer follow the flow
7. **Color contrast** - ensure elements stand out from background
8. **Loop flow** - make the feedback cycle (Scene 5) smooth and intuitive
