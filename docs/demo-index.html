📚 About These Demos

These interactive HTML animations are designed to explain annotron's core concepts visually. Each demo uses:

🎨 Hand-drawn Style Cute icons and natural animations that feel friendly and approachable
💻 Canvas-based Pure JavaScript and HTML5 Canvas — no external dependencies, lightweight
⏱️ Self-paced Play, pause, reset — take your time to understand each step
📱 Responsive Works on desktop, tablet, and mobile devices

🎯 How to Use These Demos

Option 1: Full Story — Start here if you want a complete overview. Watch all 7 scenes (90 seconds) that explain annotron from problem to solution.

Option 2: Click to Comment — If you want to understand the core interaction. Click the highlighted text in the browser window and see the structured feedback form appear in real-time.

Option 3: Loop Engineering — If you want to see how the feedback loop works. Watch as annotations flow from browser → agent → terminal → file → back to browser, completing in seconds.

Each demo has controls: Play, Pause, and Reset — use them to explore at your own pace.
