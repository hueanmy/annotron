## Video Editing Software & Steps

### Option 1: DaVinci Resolve (Free)
1. Import source video (docs/annotron-demo-v2.mp4)
2. Add voiceover track (ElevenLabs MP3)
3. Add text overlays (titles, captions)
4. Color grade: Purple/teal theme
5. Add transitions between scenes
6. Export as MP4 (H.264, YouTube preset)

### Option 2: Adobe Premiere Pro (Paid)
- Same steps, more polish options
- Better for motion graphics

### Option 3: Filmora / CapCut (Beginner-friendly)
- CapCut: Free, mobile/desktop, templates available
- Filmora: User-friendly, good effects library

### Option 4: FFmpeg (Command-line)
```bash
# Combine video + audio
ffmpeg -i annotron-demo-v2.mp4 -i voiceover.mp3 \
  -c:v libx264 -c:a aac -shortest annotron-teaser-15s.mp4

# Trim to 15 seconds
ffmpeg -i annotron-teaser-15s.mp4 -ss 0 -t 15 \
  -c:v copy -c:a copy annotron-teaser-final.mp4
```
