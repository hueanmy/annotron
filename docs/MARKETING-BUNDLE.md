## 🚀 Quick Start: From Zero to Upload

### Week 1: Generate Assets (2-3 hours)

**Day 1 - Scripts**
- [ ] Choose script from `tiktok-script-vi.md` (recommend 15s version)
- [ ] Review & customize (change YouTuber name if needed)

**Day 2 - Voiceover**
- [ ] Sign up for ElevenLabs (free tier)
- [ ] Paste script into ElevenLabs
- [ ] Generate with voice "Adam" or "Aria"
- [ ] Download MP3 → Save to `docs/voiceover-vi.mp3`

**Day 3 - Music & Video Edit**
- [ ] Find royalty-free background music (20-30s, upbeat)
- [ ] Install DaVinci Resolve or CapCut
- [ ] Follow `VIDEO-CREATION-GUIDE.md`
- [ ] Edit video (30-60 minutes)
- [ ] Export as `annotron-teaser-15s-vi-final.mp4`
- [ ] Test on phone (check audio sync, text size)

### Week 2: Upload & Promote (1-2 hours)

**Day 4 - Upload**
- [ ] TikTok: Upload video + caption from `social-media-captions.md`
- [ ] YouTube Shorts: Upload same video
- [ ] Instagram Reels: Upload to Instagram

**Day 5 - Cross-Promote**
- [ ] Tweet with video link (use caption from Twitter section)
- [ ] LinkedIn post (use professional caption)
- [ ] GitHub discussion link to YouTube

**Day 6 - Monitor**
- [ ] Check TikTok/YouTube analytics
- [ ] Respond to comments
- [ ] Note engagement metrics

## 🎯 Platform Strategy

### TikTok (Primary)
- **Frequency:** 3-5 videos/week
- **Best time:** 7-9 AM, 12-1 PM, 6-8 PM (Vietnam time)
- **Goal:** Reach tech audience, build community
- **Script length:** 15s (highest completion rate)
- **Tone:** Casual, hook-driven

### YouTube Shorts (Secondary)
- **Frequency:** 2x/week (repurpose TikTok)
- **Description:** Link to full demo + GitHub
- **Goal:** Drive to main YouTube channel
- **Script:** Same as TikTok (format vertically)

### Instagram Reels (Bonus)
- **Frequency:** 2x/week
- **Captions:** More professional than TikTok
- **Goal:** Reach professionals, designers
- **Script:** Problem-solution angle

### Twitter/X (Awareness)
- **Frequency:** 2-3x/week
- **Type:** Threads, quotes, links
- **Goal:** Developer community, press coverage
- **Style:** Technical, thought-leadership

### LinkedIn (B2B)
- **Frequency:** 1x/week
- **Type:** Long-form posts, thought leadership
- **Goal:** Enterprise audience, partnerships
- **Angle:** Productivity, AI collaboration

### GitHub (Community)
- **Frequency:** 1x (launch)
- **Type:** Discussions, Releases
- **Goal:** Developer adoption
- **Content:** Link to videos, use cases

## 💡 Pro Tips

1. **Timing matters:** Post when your audience is online
2. **Hook in first 2 seconds:** TikTok algorithm watches early engagement
3. **Captions help:** 80% of TikTok watched without sound
4. **Consistency beats virality:** Regular posting > one viral video
5. **Test & iterate:** Track what works, double down on winners
6. **Respond to comments:** High engagement signals boost algorithm
7. **Cross-promote:** Link between platforms (TikTok → YouTube → GitHub)
