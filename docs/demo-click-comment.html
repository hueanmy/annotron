Annotron is a review editor for agent-generated artifacts. When your coding agent creates HTML plans, diagrams, or mockups, giving feedback is usually clumsy.
With annotron, you simply click. Click any element or text to comment. The agent gets precise, structured feedback and applies changes in seconds.
Watch the agent work live, see updates instantly, and repeat until you're happy. It's loop engineering for artifacts — point at exactly what to change, and watch it happen in real time.

🚀 Why This Works

Traditional feedback: "Can you make the header bigger and add more contrast?"
Agent guesses → You repeat → Loop is slow.

With annotron: Click the header → add note "Bigger, higher contrast"
Agent gets: selector, text, note, images → applies in 3 seconds → you see it live.

Result: Feedback loops close in 3-5 seconds instead of minutes. You stay in flow. The agent gets precise direction.
